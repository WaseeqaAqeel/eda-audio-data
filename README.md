# DAIC-WOZ Audio Data Analysis

This repository provides tools for exploratory data analysis (EDA) on audio data from the DAIC-WOZ dataset. The analysis includes accessing audio files from zip archives, calculating summary statistics, visualizing waveforms and spectrograms, performing noise reduction, and extracting audio features.
